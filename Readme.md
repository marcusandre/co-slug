
# co-slug [![Build Status](https://travis-ci.org/marcusandre/co-slug.png?branch=master)](https://travis-ci.org/marcusandre/co-slug)

  Create a slug with co and generators.

## Installation

```
$ npm install co-slug
```

## Example

```js
var result = yield slug('make me a slug');
```

## Tests

```
$ make test
```
